# Telco Churn: Why Customers Keep Leaving (And How to Fix It)

Hey! 👋 I analyzed 7,043 telco customers and found something crazy...

## 😱 The Big Problem
- **85% of month-to-month customers churn** (that's 1,497 out of 1,745!)
- Month-to-month + no tech support = **95% leave** 
- We're losing **$2.1M revenue yearly** from customers we could save

## 🕵️‍♀️ How I Found This
1. Started with basic data check (7043 customers, 26.5% churn)
2. Contract type → Month-to-month customers disappearing
3. Cross-checked tech support → 95% risk combo found!
4. Built Excel dashboard for execs to see the problem clearly

## 💡 What I'd Tell the CEO
1. **Offer 1-year contracts** to month-to-month customers (80% churn drop)
2. **Free tech support** for fiber optic users  
3. **Win-back bonuses** for premium customers ($80+/month)

## 🎓 What I Learned
- **Start simple** - Overall churn rate first
- **Segment everything** - Contract type was the goldmine  
- **Think business** - $2.1M impact gets attention
- **Visuals win** - 85% bar chart > 1000 word
